# Коридор (Corridor)

Веб-игра «Коридор» — настольная игра в духе [Quoridor](https://ru.wikipedia.org/wiki/Quoridor).
Нужно довести свою фишку до противоположного края доски 9×9, расставляя стены и
мешая сопернику.

Планируемые режимы: **2 игрока локально** и **игра против ИИ-бота**.

## Технологии

- Python 3.14
- Flask (веб-сервер) + HTML / Canvas / JavaScript (интерфейс)
- Игровая логика — отдельный модуль на чистом Python

## Установка и запуск

```bash
# 1. Создать виртуальное окружение (один раз)
python3 -m venv .venv

# 2. Активировать его
source .venv/bin/activate        # macOS / Linux

# 3. Установить зависимости
pip install -r requirements.txt
```

> Запуск приложения появится здесь, когда будет готов код игры.
